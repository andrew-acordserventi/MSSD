Startup Instructions

1. Unzip all files to your XAMP htdocs folder (or wherever your local server files are stored).
2. Create new sql database named 'pharmacy'.
3. Create new user 'admin' and password 'admin'
4. Import database_init.sql to your php structure to build initial database.
5. Open http://127.0.0.1/project/index.php to begin.

Note, initial values for database access can be changed in the database.php file. 
These include changing the ip address, the database name ('pharmacy' by default) or the username/password.
